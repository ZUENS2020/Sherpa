# 运行并部署你的 AI Studio 应用

这里包含了在本地运行该应用所需的全部内容。

在 AI Studio 中查看此应用：<https://ai.studio/apps/88a896fa-b9cf-4302-b5d3-652a70cbcc54>

## 本地运行

**前置条件：** Node.js

1. 安装依赖：
   `npm install`
2. 在 [.env.local](.env.local) 中把 `GEMINI_API_KEY` 设置为你的 Gemini API Key
3. 启动应用：
   `npm run dev`
