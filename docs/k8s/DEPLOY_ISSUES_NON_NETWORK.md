# 部署问题（历史记录）

本文件仅作为过去部署故障的历史背景保留，不是当前运行手册。

## 历史共性问题

过去的部署事故通常来自以下几类原因：

- 运行时镜像缺少必需工具
- worker 假设与真实执行模型逐渐偏离
- 长时间运行的阶段循环掩盖了工作流逻辑问题
- 产物或输出路径与控制面预期不一致

## 如何使用本文件

如果你在排查当前部署问题，请先阅读：

- [Runbook](RUNBOOK.md)
- [Deploy](DEPLOY.md)
- [Deployment Detailed](DEPLOYMENT_DETAILED.md)

只有当你需要把新事故与旧事故做根因对比时，再回到本文。
